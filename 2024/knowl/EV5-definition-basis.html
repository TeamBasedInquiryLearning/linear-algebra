Definition 2.5.5.

A basis of a vector space \(V\) is a set of vectors \(S\) contained in \(V\) for which
  1. Every vector in the vector space can be expressed as a linear combination of the vectors in \(S\text{.}\)
  2. For each vector \(\vec{v}\) in the vector space, there is only one way to write it as a linear combination of the vectors in \(S\text{.}\)
These two properties may be expressed more succintly as the statement "Every vector in \(V\) can be expressed uniquely as a linear combination of the vectors in \(S\)".
