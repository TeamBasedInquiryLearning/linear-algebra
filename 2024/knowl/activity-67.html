Activity 2.5.2.

Consider the following set of ingredients:
\begin{equation*} S=\left\{\textrm{tomato}, \textrm{olive oil}, \textrm{dough}, \textrm{cheese}, \textrm{pizza sauce}, \textrm{garlic}\right\} \end{equation*}
(a)
Does "pizza" live inside of \(\vspan(S)\text{?}\)
(b)
Identify which ingredients in \(S\) make the set linearly dependent.
(c)
Can you think of a subset \(S'\) of \(S\) that is linearly independent and for which "pizza" is still in \(\vspan{S'}\text{?}\)
